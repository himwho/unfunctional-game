Meshy Unity Plugin - Bridge Edition
====================================

Version: 0.2.0
License: GPL-3.0

DESCRIPTION
-----------
A streamlined Unity plugin that enables direct 3D model import from Meshy.ai 
web interface to Unity Editor through a local bridge server.

FEATURES
--------
- Direct import from Meshy.ai website
- Support for GLB, FBX, and ZIP formats
- Automatic material and texture setup
- Animation clip detection and controller generation
- One-click operation

GETTING STARTED
---------------
1. In Unity Editor, go to "Meshy > Bridge"
2. Click "Run Bridge" to start the server
3. Visit Meshy.ai and generate your 3D models
4. Click "Send to Unity" on the website
5. Models automatically import to Assets/MeshyImports/

REQUIREMENTS
------------
- Unity 2021.3 or higher
- Internet connection
- Meshy account (https://www.meshy.ai)
